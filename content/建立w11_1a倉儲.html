建立w11_1a倉儲

在Github帳號下點擊 New repository

Repository name設為w11_1a

點下Create repository

打開cmd(黑色頁面)

輸入md w11_1a

輸入cd w11_1a

輸入git config --global init.defaultBranch main(利用 git init 建立的倉儲都以 main 為主要分支, 可以將設定寫入 .gitconfig)

之後便照網頁顯示的輸入

echo "# w11_1a" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/41023125/5.git
git push -u origin main


git push -u origin main之前請先設好SSH或Personal access tokens(碼牌)
輸入git submodule add https://github.com/mdecourse/cmsimde.git cmsimde
當日筆記cp2021_hw/w11notes.txt at main · mdecourse/cp2021_hw (github.com)
